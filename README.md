## Requirement

- Java 19

## Build

```bash
git clone https://github.com/EchoInMirror/EchoInMirror.git

cd EchoInMirror

gradlew :daw:shadowJar
```

## Run

```bash
java --enable-preview -jar daw/build/libs/daw.jar
```
